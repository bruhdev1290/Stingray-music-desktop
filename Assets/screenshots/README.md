# Screenshots

This directory contains screenshots of the Stingray Music Desktop application.

## Required Screenshots

Please add the following screenshots to properly showcase the application:

1. **Main Interface** (`main-interface.png`) - The primary music player interface showing the now playing screen
2. **Library View** (`library-view.png`) - The music library with albums and songs
3. **Settings** (`settings.png`) - The settings/preferences panel
4. **Themes** (`themes.png`) - Theme selection showing available themes including the new Koel theme
5. **Player Controls** (`player-controls.png`) - Close-up of playback controls and features

## Image Guidelines

- Format: PNG or JPEG
- Resolution: At least 1920x1080 for desktop screenshots
- Include various UI states (light theme, dark theme, different views)
- Ensure no personal information is visible
- Use high-quality, clear screenshots that showcase the features
