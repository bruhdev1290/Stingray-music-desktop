<h1 align="center">
Stingray Music Desktop
</h1>

This repository contains the source code for **Stingray Music Desktop**, a music client for Stingray Music built using [Electron.js](https://electronjs.org), [Vue.js 2](https://vuejs.org), and [Webpack](https://webpack.js.org).  

This is a desktop client that provides a dedicated interface for streaming music from Stingray Music.
